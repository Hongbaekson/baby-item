# 이은이 아빠가 준비하는 육아템 비주얼 디자인 기준

## 목표

`이은이 아빠가 준비하는 육아템`은 아이가 보는 것처럼 밝고 아기자기하지만, 실제 구매 링크를 빠르게 찾는 부모가 쓰기 편해야 한다. 장식은 부드럽게 쓰고, 제품 탐색 흐름을 방해하지 않는다.

## 톤

- 밝은 파스텔 톤
- 둥근 카드와 부드러운 그림자
- 카테고리마다 다른 색감의 칩
- 큰 배경 장식보다 제품 카드와 필터가 주인공인 화면
- 귀여운 스티커 느낌은 카테고리 아이콘, 플레이스홀더 이미지, 작은 배지에만 사용

## 레이아웃

- 첫 화면은 바로 제품 탐색으로 시작한다.
- 상단에는 브랜드명, 짧은 설명, 검색창, 카테고리 필터를 배치한다.
- 제품 카드는 모바일 1열, 태블릿 2열, 데스크톱 3열 이상으로 확장한다.
- 제품 상세는 별도 페이지보다 모달이 우선이다.
- 가격 영역은 `구매처에서 최신가 확인`을 기본 CTA로 표시한다.
- Notion 기준 가격이 있는 제품은 `기록가`로만 보조 표시하고, 현재 가격처럼 보이지 않게 한다.
- 검증된 `bestOffer`가 있으면 가격 영역은 `최저가`를 표시하고 `보러가기` 버튼은 `최저가 보기`로 바꾼다.
- 상품 이미지는 `bestOffer.imageUrl` 또는 원본 상품 썸네일 URL을 우선 사용하고, 없을 때만 카테고리 placeholder를 사용한다.
- 검수 필요 제품은 일반 사용자 화면에 노출하되, `정보 확인 중` 배지를 표시한다.

## 컴포넌트 기준

- 검색창: 둥근 pill 형태, placeholder는 짧게.
- 카테고리 필터: 색이 다른 chip, 선택 상태는 굵은 테두리와 체크 아이콘.
- 제품 카드: 8px 반경을 기준으로 하고, 카드 안의 정보 밀도는 유지한다.
- 구매 버튼: 눈에 띄는 코랄 계열, 외부 링크 아이콘 포함.
- 최저가 후보가 검증된 제품은 카드와 모달에 판매처, 배송비 포함 여부, 품절 아님 확인 상태를 짧게 표시한다.
- 상세 모달: 이미지, 제품명, 카테고리, 최신가 확인 CTA, 기록가, 메모, 모든 구매 링크를 표시한다.

## 이미지 기준

- 원본 제품 이미지가 있으면 제품 이미지 영역에 사용한다.
- 원본 이미지가 없으면 카테고리별 파스텔 SVG 플레이스홀더를 사용한다.
- 플레이스홀더는 젖병, 달, 유모차, 장난감, 기저귀 등 단순한 시각 힌트만 제공한다.
- 이미지가 없어도 카드 높이가 흔들리지 않도록 고정 비율을 유지한다.

## 접근성과 사용성

- 본문 텍스트는 14px 이상.
- 버튼 텍스트는 모바일에서도 줄바꿈 없이 들어가도록 한다.
- 가격 CTA처럼 긴 문구는 카드 폭 안에서 자연스럽게 줄바꿈되도록 한다.
- 색상만으로 상태를 구분하지 않고 배지 텍스트를 함께 표시한다.
- 제휴 링크 고지는 화면 하단과 상세 모달 근처에 한 번 더 노출한다.
